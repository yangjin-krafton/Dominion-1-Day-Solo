# 도미니언: 1일 솔로 (Dominion 1-Day Solo)
> 도미니언 기본 규칙은 그대로.  
> 혼자서도 “덱이 돌아가는 맛(엔진/콤보/최적화)”을 매일 한 판씩 가볍게 즐기는 데일리 솔로 챌린지.

- 🎮 GitHub Pages에서 바로 실행: `https://<YOUR_ID>.github.io/<REPO>/`
- 🧩 오늘의 10장 킹덤 + 3개 미션 + (선택) 그림자 라이벌(봇)  
- 🧠 초보자도 “성공 경험”을 느끼도록 **미션/업적 중심** 설계

> ⚠️ 이 프로젝트는 팬메이드입니다. Rio Grande Games / Donald X. Vaccarino와 무관합니다.  
> 카드 텍스트/이미지 등 도미니언 IP는 권리자에게 있습니다. 이 저장소는 **동반자 웹앱(세팅/미션/로그/점수표)** 을 목표로 하며, 기본적으로 카드 텍스트/이미지는 포함하지 않습니다.

## 이 프로젝트가 해결하는 것
도미니언의 재미는 “내 덱이 바뀌면서, 다음 턴이 달라지는 것”입니다.

혼자 할 때 더 진하게 느껴지는 포인트는 이거예요:
1. **덱 다이어트(트래싱/정리)** → 같은 5장 패라도 퀄리티가 달라짐
2. **드로우/액션/구매의 엔진화** → “한 턴에 여러 장/여러 액션/여러 구매”가 터짐
3. **타이밍의 게임(언제 그린으로 전환?)** → ‘지금 성장’ vs ‘지금 점수’
4. **콤보/시너지 퍼즐** → 오늘 주어진 10장으로 최적 루트 찾기

그래서 우리는 “상대 AI를 완전 구현”하기보다,
- 매일 바뀌는 **덱빌딩 퍼즐(오늘의 10장)**
- 초보도 달성 가능한 **3개 미션**
- (선택) 게임을 적당히 조여주는 **그림자 라이벌**
로 “도미니언 핵심 재미”를 뽑아내는 것을 목표로 합니다.

## 도미니언 규칙 리마인드 (기본 규칙 유지)
> 자세한 규칙은 공식 룰북/정식 자료를 참고하세요.

### 기본 세팅(요약)
- **기본 더미 7개**: Copper / Silver / Gold / Estate / Duchy / Province / Curse
- **킹덤 더미 10개**를 선택(게임마다 바뀜)
- 각 플레이어 시작 덱: **Copper 7장 + Estate 3장**, 섞고 5장 드로우

### 턴 순서(ABC)
1. **Action**: 기본 1장 Action 플레이
2. **Buy**: Treasure를 원하는 만큼 플레이 → 기본 1회 구매
3. **Cleanup**: 사용/손패 전부 버리고 5장 드로우(필요 시 섞기)

### 종료 & 점수
- **Province 더미가 비거나**, **공급 더미 3개가 비면** (어떤 더미든) 그 턴 종료 후 게임 종료
- 내 덱(덱/버림/손/플레이/세트어사이드 포함)의 Victory Point 합산

## 솔로 모드
### 모드 A — **그림자 라이벌(추천)**
“상대의 심리전” 대신, **공급이 줄어드는 압박 + 종료 타이밍**을 만들어주는 초간단 봇입니다.

#### 준비
- 공급 더미 수량은 **2인 기준**으로 추천(실질적으로 ‘나 + 라이벌’ 2인 게임 속도)
- 라이벌용 준비물: **Budget 덱(코인 덱)** 1개

#### Budget 덱(기본/노말)
아래 숫자를 카드/메모로 만들어 섞어두세요 (총 16장):
- 0×1, 2×2, 3×2, 4×2, 5×3, 6×3, 7×2, 8×1

> 쉬움: 8을 제거, 0을 1장 추가  
> 어려움: 8을 1장 추가(총 17장) 또는 7을 8로 1장 업그레이드

#### 라이벌 턴(내 Cleanup 직후 1번 실행)
1. Budget 덱에서 1장 공개 → 이 값이 이번 라이벌의 “구매력”
2. 라이벌은 **공급에서 카드 1장**을 획득(라이벌 영역에 놓기)  
   우선순위(위에서부터 가능한 첫 항목):
   1) Province (비용 8)  
   2) Gold (6)  
   3) Duchy (5)  
   4) “가장 비싼” Kingdom 카드(비용 ≤ Budget)  
      - 동률이면: 공급에 놓인 순서(왼쪽→오른쪽)로 첫 번째
   5) Silver (3)  
   6) Estate (2)  
   7) Copper (0)

3. Budget 덱이 다 떨어지면 다시 섞습니다.

#### 공격 카드 처리(간단 버전)
- “상대가 저주를 얻는다”류는 **라이벌에게 그대로 적용**(예: Witch → 라이벌 Curse 1장 획득)
- 그 외 “손패/덱을 건드리는 공격”은 **무시**(라이벌은 추상화된 존재)

#### 승리 조건
- 기본 종료 조건으로 게임이 끝나면,
  - 내 VP vs 라이벌 VP 비교
  - 동점이면 “내가 얻은 Province 수”가 많은 쪽 승리
  - 그래도 동점이면 무승부(오늘의 미션 점수로 만족!)

## 미션: 초보도 ‘성공’을 느끼게 하는 장치
승패와 별개로, 아래 같은 “덱빌딩 감각”을 미션으로 보상합니다.

### 미션 예시(웹앱에서 3개 랜덤 선택)
- 🧹 덱 다이어트: 게임 중 시작 카드(구리/장원) 합쳐서 4장 이상 줄이기(트래시/업그레이드 포함)
- 🔁 순환 가속: 한 턴에 카드 8장 이상 보기(드로우/서치 포함)
- 🧠 액션 체인: 한 턴에 Action 카드 3장 이상 플레이
- 🪙 경제 폭발: 한 턴에 코인 8 이상 만들기
- 🛒 멀티 구매: 한 턴에 2장 이상 구매(추가 구매 확보)
- 🏁 전환 타이밍: Province를 “첫 1장” 얻은 턴을 기록하고, 다음 판에 1턴 앞당기기
- 🎯 엔드게임 컨트롤: 게임 종료를 “내가 의도한 턴”에 가까이 맞추기(±1턴)

### 점수(별) 추천
- 각 미션 달성: ⭐ 1
- 라이벌에게 승리: ⭐ 추가 1
- Province 4장 이상: ⭐ 추가 1

> 초보자는 “별 2개”만 목표로 해도 충분히 재미가 납니다.  
> 숙련자는 “별 4~5개”를 노리면서 최적화 퍼즐로 즐기면 됩니다.

## 초보자 3판 코스 (추천)
1) **기본 흐름 익히기**  
   - 목표: “코인 8 만들기”를 최소 1번
2) **덱 다이어트 맛보기**  
   - 목표: 시작 카드 3~5장 정리 + Action 2장 연속 플레이
3) **엔진 첫 성공**  
   - 목표: “한 턴에 Action 3장 + 구매 2번” 또는 Province 2장

> 첫 3판은 “이기기”보다, 덱이 빨라지고 강해지는 감각을 잡는 게 더 중요합니다.

## GitHub Pages로 배포하기 (HTML)
추천 구조:
```

/docs
index.html
styles.css
app.js
README.md

```

1) 저장소 `Settings` → `Pages`
2) Source를 `Deploy from a branch`로 선택
3) Branch: `main` (또는 사용하는 브랜치)
4) Folder: `/docs`
5) `Save` 후 배포 URL 확인
